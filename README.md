# 🛍️ Customer Segmentation using RFM Analysis and Machine Learning

## 📌 Project Overview

This project performs customer segmentation using RFM (Recency, Frequency, Monetary) analysis combined with machine learning techniques. 

The system groups customers into meaningful clusters and builds predictive models to classify new customers based on purchasing behavior.

An interactive web application is developed using Streamlit to visualize clustering results and model performance.

---

## 🔵 Unsupervised Learning

Two clustering algorithms are implemented:

### 1️⃣ KMeans Clustering
- Dynamic selection of number of clusters
- Elbow Method visualization
- Silhouette Score evaluation
- Cluster visualization graph

### 2️⃣ Agglomerative Clustering
- Hierarchical clustering approach
- Silhouette Score comparison
- Cluster visualization graph

These techniques help identify distinct customer groups based on RFM features.

---

## 🟢 Supervised Learning

Cluster labels generated from KMeans are used to train classification models.

### 1️⃣ Logistic Regression
- Accuracy
- Precision
- Recall
- F1 Score
- Confusion Matrix

### 2️⃣ Random Forest
- Accuracy
- Precision
- Recall
- F1 Score
- Confusion Matrix

A comparison graph is included to evaluate model performance.
